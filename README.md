## Running on Local Machine
1. Download this repository
2. Run api.py
    <br /> &emsp; $ python /src/api.py
3. Run streamlit.py
    <br /> &emsp; $ streamlit run /src/streamlit.py
4. Open http://localhost:8501 in browser
5. Enter variables, then click predict
